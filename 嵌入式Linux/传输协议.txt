								IIC
zzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzz
【一】特点
	1.是串行总线
	2.I2C串行总线一般有两根信号线，一根是双向的数据线SDA，另一根是时钟线SCL。
	3.I2C总线上允许连接多个微处理器以及各种外围设备，如存储器、LED及LCD驱动器、A/D及D/A转换器等
	4.为了保证数据可靠地传送，任一时刻总线只能由某一台主机控制
	5.每一支IIC设备都有一个唯一的七位设备地址；
	6.数据帧大小为8位的字节；
	7.数据（帧）中的某些数据位用于控制通信的开始、停止、方向（读写）和应答机制。
	8.IIC 数据传输速率有标准模式（100 kbps）、快速模式（400 kbps）和高速模式（3.4 Mbps）
	
【二】数据传输
	1.IIC协议标准规定发起通信的设备称为主设备
	
	2."Start": SCL为高电平时, SDA由高电平变为低电平
	  "Stop" : SCL为高电平时, SDA由低电平变为高电平
	3.数据传输流程
		1.首先，主设备发一个START信号，这个信号就像对所有其它设备喊：
			请大家注意！然后其它设备开始监听总线以准备接收数据
		2.接着，主设备发送一个7位设备地址加一位的读写操作的数据帧	
		3.当所设备接收数据后，比对地址自己是否目标设备。
			如果比对不符，设备进入等待状态，等待STOP信号的来临；
			如果比对相符，设备会发送一个应答信号――ACKNOWLEDGE作回应	
		4.当主设备收到应答后便开始传送或接收数据。数据帧大小为8位，尾随一位的应答信号。
		5.主设备发送数据，从设备应答；相反主设备接数据，主设备应答。
		6.当数据传送完毕，主设备发送一个STOP信号，向其它设备宣告释放总线，其它设备回到初始状态。
	4.SDA线的数据转换必须在SCL线的低电平期

								SPI
zzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzz
【一】特点
	1.SPI是串行外设接口(Serial Peripheral Interface)的缩写。是 Motorola 公司推出的一 
		种同步串行接口技术，是一种高速的，全双工，同步的通信总线。
	2.优点
		支持全双工通信
		通信简单
		数据传输速率块
	3.缺点
		没有指定的流控制，
		没有应答机制确认是否接收到数据，所以跟IIC总线协议比较在数据可靠性上有一定的缺陷。	
	4.主从机通信模式
	
【二】数据传输
	1.SPI的通信原理很简单，它以主从方式工作，这种模式通常有一个主设备和一个或多 个从设备，需要至少4根线
	2.引脚
		SDO/MOSI C 主设备数据输出，从设备数据输入;
		SDI/MISO C 主设备数据输入，从设备数据输出;
		SCLK 	 C 时钟信号，由主设备产生;
		CS/SS 	 C 从设备使能信号，由主设备控制。当有多个从设备的时候，因为每个从设 
				备上都有一个片选引脚接入到主设备机中，当我们的主设备和某个从设备通信时将需 
				要将从设备对应的片选引脚电平拉低或者是拉高
	3.SPI通信有4种不同的模式
			（时钟极性）（时钟相位）
		Mode0：CPOL=0，CPHA=0
		Mode1：CPOL=0，CPHA=1
		Mode2：CPOL=1，CPHA=0
		Mode3：CPOL=1，CPHA=1
		时钟极性： 控制是高电平，还是低电平采集
		时钟相位： 控制是上升沿，还是下降沿采集

								IIS
zzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzz
I2S有3个主要信号：
	1.串行时钟 SCLK，也叫做位时钟BCLK，即对应数字音频的每一位数据，SCLK的频率=2×采样频率×采样位数 	
	2.帧时钟LRCK，用于切换左右声道的数据，LRCK为“0”表示正在传输的是左声道的数据，为“1”表示正在传输的是右声道的数据。LRCLK == FS,就是采样频率 
	3.串行数据SDATA

								UART
zzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzzz
UART是一种通用串行数据总线，用于异步通信。该总线双向通信，可以实现全双工传输和接收。在嵌入式设计中，UART用于主机与辅助设备通信
	工作原理是将传输数据的每个字符一位接一位地传输。
	传输:
		起始位：先发出一个逻辑”0”的信号，表示传输字符的开始。
		数据位：紧接着起始位之后。资料位的个数可以是4、5、6、7、8等，构成一个字符。通常采用ASCII码。从最低位开始传送，靠时钟定位。
		奇偶校验位：资料位加上这一位后，使得“1”的位数应为偶数(偶校验)或奇数(奇校验)，以此来校验资料传送的正确性。
		停止位：它是一个字符数据的结束标志。可以是1位、1.5位、2位的高电平。 由于数据是在传输线上定时的，并且每一个设备有其自己的时钟，很可能在通信中两台设备间出现了小小的不同步。因此停止位不仅仅是表示传输的结束，并且提供计算机校正时钟同步的机会。适用于停止位的位数越多，不同时钟同步的容忍程度越大，但是数据传输率同时也越慢。
		空闲位：处于逻辑“1”状态，表示当前线路上没有资料传送。
		波特率：是衡量资料传送速率的指标。表示每秒钟传送的符号数（symbol）。一个符号代表的信息量（比特数）与符号的阶数有关。例如传输使用256阶符号，每8bit代表一个符号，资料传送速率为120字符/秒，则波特率就是120baud，比特率是120*8=960bit/s。这两者的概念很容易搞错。
